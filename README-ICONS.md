# Crear Iconos para PWA

La aplicación necesita iconos para funcionar como PWA. Tienes dos opciones:

## Opción 1: Usar el Generador HTML (Recomendado)

1. Abre `create-icons.html` en tu navegador
2. Haz clic en "Generar icon-192.png" y descarga el archivo
3. Haz clic en "Generar icon-512.png" y descarga el archivo
4. Coloca ambos archivos en la raíz del proyecto

## Opción 2: Crear Iconos Manualmente

Necesitas crear dos archivos PNG:
- `icon-192.png` - 192x192 píxeles
- `icon-512.png` - 512x512 píxeles

Puedes usar cualquier editor de imágenes o herramientas online como:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- Cualquier editor de imágenes (Photoshop, GIMP, etc.)

Los iconos deben tener:
- Fondo: #2d5aa0 (azul)
- Texto/Imagen: Blanco o color contrastante
- Formato: PNG con transparencia (opcional)

## Nota

Los iconos no son críticos para que la aplicación funcione, pero son necesarios para:
- Instalación como PWA
- Mejor experiencia visual en el navegador
- Agregar a pantalla de inicio (si está disponible)

Si no tienes los iconos, la aplicación seguirá funcionando normalmente desde el navegador.
